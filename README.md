# Titanic Survival Prediction

This project predicts whether a Titanic passenger survived or not using Machine Learning.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Passenger Class
- Gender
- Age
- Siblings / Spouses
- Parents / Children
- Fare

## Technologies
- Python
- Pandas
- Scikit-Learn
- Streamlit

## Run the Project

pip install -r requirements.txt

streamlit run app.py
